Static management
-----------------

You can use the ``pstatic`` command in a terminal window to copy a 
all registered static resource to specified directory.

.. code-block:: text
   :linenos:
   
   [fafhrd@... MyProject]$ ../bin/pstatic development.ini /path-to_dir/
